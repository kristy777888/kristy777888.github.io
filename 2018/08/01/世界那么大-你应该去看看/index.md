# 世界那么大 你应该去看看

&nbsp;

<span style="font-size: 12pt;">世界那么大，我想去看看，怎么看世界，其实很简单，我就纳闷了，那么多人天天找挖皮嗯，找梯子，找科学上网，自己就不能好好学一下，自己弄个酸酸乳（ssr）吗。</span>

<span style="font-size: 12pt;">世界就在那，不远不近，你想去看，就可以去看，就好比赚钱，钱就在那，看你去不去捡，捡就弯腰，不捡你就在那看，和别人有什么关系，所以这里感谢那些真正分享的朋友们，因为有了你们，这个世界才更加丰富多彩，感恩仅仅两个字，但每个阶段体会是不一样的，刚接触网赚行业的时候，大家就跟我说执行力这三个字，这么多年，每个阶段对这三个都会有更加深刻的感触。</span>

**<span style="font-size: 12pt;">先说下结果，你学会了今天的教程会怎么样？</span>**

<span style="font-size: 12pt; color: #ff6600;">1.会搭建属于自己的ssr，2.可以遨游世界了不管你是在pc端，ios端，还是安卓端。</span>

原理：

通过VPS搭建ssr，起到服务器中转作用，你通过国内工具连接到国外VPS，然后VPS访问目标网站，获取信息后中转，返回国内，从而起到观看世界的效果。

&nbsp;

<span style="font-size: 12pt;">下面就小白式，傻瓜式，手把手，嘴对嘴式教学。</span>

<span style="font-size: 12pt;">通过板瓦工，来搭建属于自己的ssr。</span>

<span style="font-size: 12pt;">如果你有vpn或这能打开下面的地址就通过下面的地址注册账号</span>

<span style="font-size: 12pt;"><strong><a href="https://bandwagonhost.com/aff.php?aff=23876">https://bandwagonhost.com/aff.php?aff=23876</a></strong></span>

<span style="font-size: 12pt;">如果访问不了就用下边的链接注册</span>

<span style="font-size: 12pt;"><a href="http://www.bwh1.net"><strong>www.bwh1.net</strong></a></span>

点击右上方的**Register注册，**开始填写具体信息。

<img loading="lazy" class="aligncenter size-large wp-image-418" src="http://www.zhangliguo.com/wp-content/uploads/2018/08/QQ截图20180802033055-1024x832.png" alt="" width="1024" height="832" /> 

2.注册号登陆，然后就是购买VPS了，根据自己的需求购买，我自己的是20G的vps KVM的，KVM,ORZ,XEN,都什么区别，我就不细说了想了解的看下面csdn一兄dei说的。购买可以使用支付宝付款，alipay就是，很方便简单。

https://blog.csdn.net/enweitech/article/details/52910082

3.然后点击中间导航Services栏目下的My Aervices，这时就会看你购买的vps了，然后点击后面的 KiwiVM Control Panel，进入Kvm的控制面板。

<img loading="lazy" class="aligncenter size-large wp-image-422" src="http://www.zhangliguo.com/wp-content/uploads/2018/08/QQ截图20180802035651-1024x498.png" alt="" width="1024" height="498" /> 

初次购买后，进入系统应该提示让你选择安装VPS的系统，不要觉得难，麻烦，就是在异地一个你的电脑，选择安装一个系统，就这么理解就好了，我选择的私centos6 x86_64 bbr，以后要换系统，点击Install new OS 选项即可。

<img loading="lazy" class="aligncenter size-large wp-image-421" src="http://www.zhangliguo.com/wp-content/uploads/2018/08/QQ截图20180802035845-1024x689.png" alt="" width="1024" height="689" /> 

4.系统安装完了，下面来安装ssr服务，点击kvm控制面板，选择左侧导航的 ShadowsocksR Server选项，点击安装，安装后就可以看见相应的ssr的配置信息了。

<img loading="lazy" class="aligncenter size-large wp-image-423" src="http://www.zhangliguo.com/wp-content/uploads/2018/08/QQ截图20180802040210-1024x746.png" alt="" width="1024" height="746" /> 

&nbsp;

下面就是，在你的电脑端，手机端，pad端安装ssr的客户端了，以后开启客户端，配置好信息，就可以起飞了，电脑端ssr。

下载地址该页面都有显示，直接点击下载就可以了，安卓和pc端的不说了，直接下载安装，然后下载下来添加服务器，然后把相应信息填进去，就好了。

下面主要说下ios的，页面给出ios的下载地址，已经不存在了

现在有两种解决方法：

一个是你用非国内的苹果id进入appstore，下载shadowsocks，然后进行相关配置即可。

另一个就是，下载pp助手，有有越狱版本和非越狱版本，看自己手机情况下载，然后搜索shadowsocks安装相关提示下载即可。
