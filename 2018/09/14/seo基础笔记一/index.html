这是前两天看电脑里的一份教程 总结出来的，虽然教程有些远久了，但是基础原理还是不变得，一般变得都是更新升级反作弊系统方面的，还有优化一下seo，所以还是值得一看的。

  • SEO 基础 1、如何正确认识 SEO

    • 大家要把 SEO 看简单一些，因为 SEO 确实没有什么技术含量，因为影响搜索引擎排名的因
      素，全部都是一系列细节因素。
    • 只有你有耐心去认真把这些细节做好，排名是很容易获得的
    • 但是，目前绝大部分人都很浮躁，甚至国内 80%的 SEO 公司，只是靠其中的一两招在混饭
      吃
    • 所以，学习 SEO 是很容易的事情
    • 可以说，绝大部分 SEO 公司都是靠我们讲的八大因素中的 反向链接因素来做排名
    • 虽然效果很明显，但是若前面的细节做不好，排名很难稳定，所以要想获得长期稳定的好排
      名，还是需要认认真真把每一个简单的细节都去花时间认真做好
    • 所以，只要认真、细心、耐心，SEO 是非常容易的

    2、影响排名的因素

    • 1、服务器因素
    • 服务器因素就这么简单，主要几点：
      1、稳定性：避免中木马，避免经常打不开
      2、速度：速度越快，排名越有利
      gogole 世界各地排名不同，做外贸网站要放在国外服务器上
    • 2、内容因素
    • 所以，网站内容越丰富，搜索引擎就会认为你越专业，就越容易获得排名

    搜索引擎是如何判断原创内容呢？

    • 1、被收录的时间的早晚
    • 2、文章中指向的链接
    • 如果你把别人的文章，修改一下，里面加上你的网站的很多链接，然后到处去转摘
    • 搜索引擎也可能把它当成你的网站的原创
    • 1、网站内容越丰富，对排名越有利
    • 2、网站原创内容越多，对排名越有利
    • 例如你在各大网站开 20 个博客
    • 然后在这些博客里面发
    • 那如果我同样写这样一篇文章 稍微修改！然后直接写在我的站？引擎怎么区别是不是我自
      己的原创？
    • 发到你的网站
    • 同时要到其他网站也发一下
    • 因为很多网站都有这篇内容，并且里面的链接链接到了你的网站
    • 所以搜索引擎就会认为是你的原创

    3、title 和 meta 因素

    titile和meta设计五个原则：

    • 原则一：一个网页的 title 和 meta 中，核心关键词越少越好
    • 因为一个网页的权重是有限的，如果你一个网页放几十个关键词，那个关键词获得的权重都
      不高，都排不上去，并且还有关键词堆积作弊的嫌疑
    • 这个是关键词标签，告诉搜索引擎，这个网页大概什么内容
    • 同时，网页的 title 和网页描述标签中的内容还出现在搜索结果索引中
    • 原则二：每个页面的 title 和 meta 标签都要不同，并且要与该页面的内容相符合
    • 原则三：title 设计越简洁明了越好，尽量不要超过 25 个汉字。不要超过 50 个英文字符
    • description 不要超过 100 个字，因为搜索引擎只能索引到两行。大概七八十个汉字最为合理
    • keywords 这个标签不是很重要，不要堆积关键词，合理放一两个该网页的关键词就 OK 了
    • 原则 4：title 和 meta 标签中关键词出现的频率
    • title 中，出现 1 次
    • 网页描述标签 3-4 次比较自然合理
    • 关键词标签，1 次就可以了
    • 原则五：title 中，关键词尽可能放前面
    • 一个网站的关键词越多越好
    • 一个网页的关键词越少越好

    4、网页排版细节

    • 1、大标题要用


      2、文本中的的关键词用加粗或者加重两三次
      3、网页中的图片要加上 alt 注释

    网站的首页关键词应该放多少合适？

    越少越好，最好不要超过 3 个

    合理加粗两三次

    用户看起来比较醒目 搜索引擎也可以识别这个关键词比较突出就可以了

    5、域名、文件名、url 因素

    • 主要得益于域名包含关键词
    • 使用拼音域名对排名的影响是非常小的
    • 拼的有些帮助，但是帮助不大，可以忽略不计
    • 但是英文的影响非常大
    • 所以，我们做英文站在选择域名的时候，要选择包含关键词的域名比较好，同时设计网页文
      件名，最好使用关键词文件名
    • 1、 WWW 前缀的二级域名权重最高
      www.xxxx.com 比 aaa. xxxx.com 权重要高
    • 2、 二级域名比目录有优势
      aaa. xxx.com 比 www. xxx.com/aaa/ 权重要高
    • 3、 目录越深对排名越不利
      www. xxx.com/aaa/比 www. xxx.com/aaa/bbb/ 权重要高
      一个网页的目录深度最好不要超过三级
    • 目录权重比一个静态网页高
      www.uxxx.com/aaa/比 www. xxx.com/aaa.htm 权重高
    • 静态 URL 比动态 URL 权重高
      www.xxx.com/aaa.htm 比 www.xxx.com/aaa.asp?a=123
    • 为什么会有这样的权重呢？
    • 因为：搜索引擎在模拟人的思维方式在判断，那个网页更加专业
    • 1、 www.xxx.com 这样顶级域名下的网页，一般都是一个网站的首页
    • 2、 aaa.xxx.com 这样二级域名下的网页，一般都是站内小网站的首页
    • 3、 www.xxx.com/aaa/ 这样的目录下的网页，一般都是顶级栏目的首页
    • 4、 www.xxx.com/aaa/bbb/ 这样的目录下的，一般都是二级栏目的首页
    • 5、 www.xxx.com/aaa.htm 这样的 URL 的网页，一般都是一个普通内页
    • 6、 www.xxx.com/aaa.asp?a=123 这样的一个 URL 的网页，一般都是批量出现的网页
    •  
    • 什么样的算知名网站，什么样的算非知名网站，搜索引擎靠什么来判断
    • 例如这些个行业知名的门户网站，他们用二级域名没事

     

    6、网站链接构架因素

    • 1、导航结构要清晰明了
      2、超链接要用文字链接
      3、各个页面要有相关链接

    因为 搜索引擎蜘蛛更新你的网站，是顺着链接爬行更新的

    如果你的每个网页都有相关网页的链接

    这样搜索引擎蜘蛛比较容易爬行。很快就把你所有网页更新完了

    7、关键词布局因素

    • 在一个网页中的六个地方把关键词布局好，密度自然合理
    • 1、Title、
      2、meta、
      3、网页大标题、
      4、网页文本、
      5、图片 alt 注释、
      6、网页中的链接包含关键词

    8、反向链接因素

    • 反向链接是指 A 网页上有一个链接指向 B 页，那么 A 页就是 B 页的反向链接。
    • 反向连接是针对网页和网页之间的 。
      不是针对网站和网站之间的，大家要明白这个
    • 站内网页之间的链接，叫内部链接
      站外网页给你的链接。叫站外链接
    • 不管站内站外，都是反向链接
    • 反向链接的多与少和质量决定你的网站的 PR 值
    • PR 值是 pages rank
    • 就是 google 对一个网页的评分
    • http://toolbar.google.com
    • 网站优化好之后，拼的就是反向链接
    • 什么是高质量的反向链接？
    • 1）、PR 值高的网页
    • 2）、更新频率高的网页
    • 3）、权威网页
    • 4）、相关性高的网页
    • 质量不高的量大，对你就有帮助
    • 有质量高的，当然就更好不过了
    • 反向链接的质量和数量决定着 PR 值的高低
    • PR 越高 说明你的网页质量越高
    • 、做反向链接，最关键最关键最关键的一点是：要用 “ 关键词 ” 来链接
    • 比如，我想提高 SEO 训 培训 这个词的排名
    • 我就需要在非常多的网页上 用 SEO 培训 这个词链接指向到我的网站
    • 所以，大家交换链接的时候不要用网站名 公司名要用关键词你想排什么词，就要用什么词来做链接
    • 如何判断他是不是作弊的网站呢？
    • 用 site：查询一下他是否被搜索引擎收录
    • 没有收录，如果又是老网站，那就是作弊被封了

主要的基本就是以上红色的几个部分，如何你注意以上的因素，再加上你的原创内容，你的网站一定会有一个好的排名，其实seo最高的境界就是没有seo，这个慢慢理解，在之前的一篇文章关于《原创内容》的大家可以看下，其实大道至简，在于的是一份坚持，和一个好的方向。
